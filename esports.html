Role and Responsibilities

As a Web Intern, I developed and managed an Esports website with a team, displaying the latest gaming news and scores in the growing Esports world.

Elevating Esports.

Defining goals, requirements, and conducing extensive user research led to a seamless website design.

A Sleek & Engaging Website Experience.

Your esports world, personalized. The dashboard brings everything you care about into one experience, giving you instant access to stats, news, and the communities that matter most. I worked on prioritizing key user needs while aligning with business objectives to maximize engagement.

Mobile Friendly.

We built the website to be mobile friendly for a clean, responsive design for a smooth user experience. We focused on easy navigation, fast performance, and engaging features across all devices.

Optimized for Every Screen.

I prioritized mobile responsiveness, defining breakpoints and ensuring a consistent, high-performance experience across all devices.

Guided by the North Star.

We focused on the North Star to guide key improvements, ensuring features and interactions enhance engagement and long-term growth.

Created with the User in Mind.

This platform was built using Agile methodologies, iterating quickly based on user feedback and data insights. This experience sharpened my ability to prioritize features, adapt to changing requirements, and drive continuous improvements with a user-first mindset.
